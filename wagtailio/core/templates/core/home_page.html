Developers

Wagtail is built by developers for developers. Spend less time on configuration and more perfecting your site.

As a Django app Wagtail is the CMS that will play nicely with everything else in your tech stack. It’s an un-opinionated toolbox to help you build your project without reinventing the wheel.

Designers

Display the content the way you want. Using Django’s simple templating system allows you to build a beautiful website the way you want to, not the way the CMS thinks you should build it.

Prefer a headless approach to work some MVC magic? That’s fine, just send the content through Wagtail’s REST API and receive a JSON package for absolute front-end freedom.

Budget Holders

Wagtail is open source. We’ve used the BSD license, meaning it’s both free from costs, and free from strings. No more monthly licensing charges!

And we’re here to support and encourage a community of developers to make Wagtail better and better. We have a roadmap of upcoming features and you can have confidence the CMS will improve with every release.

Editors

We created Wagtail to allow you to create once and publish everywhere. Content models are defined in code, ensuring that relationships are consistently defined for a zen-like content architecture.

With StreamField you can create beautiful, modular ‘streams’ of content that neatly separate presentation from content. All that, and the user-interface for content entry looks beautiful too!
